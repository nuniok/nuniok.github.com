不要使用 IDE 内嵌 Maven

基本命令

mvn compile 编译源码
mvn package 打包
mvn test 测试
mvn clean 清除代码
mvn deploy 上传到私服
mvn install 上传到本地仓库

mvn -X install 开启调试日志
mvn -U package 清除缓存打包
mvn cobertura:cobertura 单元测试覆盖率
mvn dependency:tree 查看依赖关系
